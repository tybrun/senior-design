## Project Abstract
2D virtual board game, based on the board game Air Force Wargame: Indo-Pacific

## Overview
The Air Force Wargaming Institute created Air Force Wargmae: Indo-Pacific (AFWI), a physical board game,
to train AFROTC cadets about strategic airpower decision-making within the Indo-Pacific theater. The game simulates a hypothetical conflict between the US and PRC,
requiring players to manage air, maritime, space, and cyber effects across multiple turns.
Virtual AFWI translates every mechanic of the physical game into an interactive desktop game built with Python and Pygame.
The digital system eliminates the friction of physical components — cards, tokens, boards, dice — and allows two players to engage in a complete, rules-enforced session on a single computer.
The game also features a brief tutorial that guides new players through mechanics step-by-step.
