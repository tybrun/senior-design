# Senior Design - Tyler Brunelle
This repository is for Tyler Brunelle's Senior Design project

# Project Description
Air Force Wargame: Indo-Pacific (AFWI) is a board game created by the Air Force Wargame Institute.
I am making a virtual version of this board game. This will help anyone access the game, be able to enforce all of the rules, and play the game anytime, anywhere!
